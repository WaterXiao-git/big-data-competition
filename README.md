# Big Data Competition — 大数据竞赛

> **大数据竞赛项目代码**。
> 包含 LSTM、Transformer 等深度学习模型在大数据场景下的应用实现。

---

## 📋 模型列表 | Models

| 文件 | 模型 | 说明 |
|------|------|------|
| LSTM.py | LSTM 神经网络 | 长短期记忆网络实现 |
| LSTM2.py | LSTM 变体 | 改进版本 |
| Transformer.py | Transformer | 注意力机制模型 |
| evaluate.py | 评估脚本 | 模型评估与指标计算 |
| 1.py | 工具脚本 | 数据预处理与分析 |

## 🚀 快速开始 | Quick Start

```bash
python LSTM.py      # 训练 LSTM
python evaluate.py  # 评估结果
```
